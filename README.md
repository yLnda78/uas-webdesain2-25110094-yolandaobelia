# Website Company Profile — Sarapan Pagi TKA Pekanbaru

**Nama       :** Yolanda Obelia
**Matkul     :** Web Desain 2 (MKK 123)
**UMKM       :** Sarapan Pagi TKA Pekanbaru
**Lokasi     :** Perum, Jl. Kuantan Raya Blok I No.15, Sekip, Kec. Lima Puluh, Kota Pekanbaru, Riau 28142
**Deskripsi  :** Website ini merupakan platform company profile interaktif untuk UMKM kuliner "Sarapan Pagi TKA Pekanbaru" yang menyajikan hidangan sarapan khas Tionghoa berlabel 100% Halal. Fitur-fitur utama pada website ini meliputi:
- **Home**: Menampilkan banner utama, nilai keunggulan (Hygienis, Rasa Otentik, Pelayanan Gesit), dan sekilas profil usaha.
- **Daftar Menu Hidangan**: Katalog menu sarapan seperti Bakmie Karet, Bihun Kuah, dan Kwetiau Kuah lengkap beserta penjelasan kehalalan dan topping.
- **Reservasi Kursi & Meja**: Fitur interaktif untuk memilih layout lokasi meja secara visual (Porsi Keluarga & Porsi Santai) serta form pemesanan.
- **Tentang Kami**: Informasi sejarah, komitmen higienitas, daftar fasilitas warung, serta jam operasional bisnis.
- **Kontak & Lokasi**: Form kirim masukan terintegrasi ke WhatsApp, integrasi Google Maps interaktif, serta tautan ke media sosial resmi (@sarapanpagi_tka).
